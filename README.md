#  Health Dashboard Web App (Flask)

A multi-page health analytics web application built with Flask.  
It simulates a lightweight personal health monitoring system inspired by Apple Health UI design.

---

##  Features

###  Sleep Analysis Module
- Sleep timeline visualization (24-hour cycle)
- Inputs: bed time, sleep time, wake time, get-up time, night awakenings
- Sleep duration calculation
- Sleep quality scoring system
- Automated health comments (e.g. late sleep, early wake, frequent awakenings)

---

###  Advanced Health Module
- BMI calculation (height & weight)
- Blood pressure evaluation (optional input)
- Blood glucose evaluation (optional input)
- Risk classification system (Good / Warning / Danger)
- Overall health score

---

### UI Design
- Apple Health–inspired card-based interface
- Color-coded risk system:
  - 🟢 Good
  - 🟡 Warning
  - 🔴 Danger
- Clean, minimal dashboard layout
- Responsive multi-page navigation

---

## Tech Stack

- Python (Flask)
- HTML / CSS
- Jinja2 templating
- Rule-based scoring logic
